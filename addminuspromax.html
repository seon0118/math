🔧 管理員設定指南

第一步：建立 Google Sheets

  1. 開啟 Google Sheets 建立新試算表
  2. 將試算表命名為「正負數競賽成績記錄」
  3. 系統會自動建立標題列，無需手動設定

第二步：設定 Google Apps Script

  1. 在試算表中點選「擴充功能」→「Apps Script」
  2. 刪除預設的 myFunction，貼入以下程式碼：
function doPost(e) {
  try {
    // 取得試算表
    const sheet = SpreadsheetApp.getActiveSheet();
    
    // 解析傳入的資料
    const data = JSON.parse(e.postData.contents);
    
    // 如果是第一次使用，加入標題列
    if (sheet.getLastRow() === 0) {
      sheet.getRange(1, 1, 1, 9).setValues([
        ['學校', '姓名', '班級', '總題數', '答對題數', '正確率', '平均時間', '遊戲時間', '系統版本']
      ]);
      
      // 設定標題列格式
      const headerRange = sheet.getRange(1, 1, 1, 9);
      headerRange.setBackground('#4285f4');
      headerRange.setFontColor('#ffffff');
      headerRange.setFontWeight('bold');
    }
    
    // 新增資料到下一列
    const nextRow = sheet.getLastRow() + 1;
    sheet.getRange(nextRow, 1, 1, 9).setValues([[
      data.schoolName || '未設定',
      data.name,
      data.class,
      data.totalQuestions,
      data.correctAnswers,
      data.accuracy,
      data.averageTime,
      data.gameTime,
      data.systemVersion || '1.0'
    ]]);
    
    // 設定資料格式
    const dataRange = sheet.getRange(nextRow, 1, 1, 9);
    dataRange.setBorder(true, true, true, true, true, true);
    
    return ContentService
      .createTextOutput(JSON.stringify({ success: true }))
      .setMimeType(ContentService.MimeType.JSON);
      
  } catch (error) {
    return ContentService
      .createTextOutput(JSON.stringify({ success: false, error: error.toString() }))
      .setMimeType(ContentService.MimeType.JSON);
  }
}

第三步：部署應用程式

  1. 點選「部署」→「新增部署作業」
  2. 類型選擇「網頁應用程式」
  3. 說明填入：「正負數競賽成績收集系統」
  4. 執行身分：選擇您的帳戶
  5. 存取權限：選擇「任何人」
  6. 點選「部署」

第四步：配置系統

  1. 複製部署後的網址
  2. 在程式碼中找到「系統配置區域」
  3. 將 YOUR_SCRIPT_ID 替換為您的完整網址
  4. 可選：修改學校名稱、科目名稱等設定
