【基礎核心】三角形外角定理與內角和

所有模型的推導基礎：
1. 三角形的一個外角，等於其兩個「內對角」的和。
2. $\triangle$ 內角和 $= 180^\circ$。

一 8字模型系列

模型 1：基本 8 字
$\angle 1 + \angle 2 = \angle 3 + \angle 4$
📝 展開證明
在 $\triangle AOB$ 中：$\angle 1 + \angle 2 + \angle AOB = 180^\circ$
在 $\triangle COD$ 中：$\angle 3 + \angle 4 + \angle COD = 180^\circ$
又 $\because \angle AOB = \angle COD$ (對頂角相等)
$\therefore \angle 1 + \angle 2 = \angle 3 + \angle 4$
模型 2：8字 + 角平分線
$\angle P = \frac{\angle B + \angle C}{2}$
📝 展開證明
由 8 字模型可得：
$\angle B + \angle \alpha = \angle P + \angle \beta \Rightarrow \angle \alpha - \angle \beta = \angle P - \angle B$
$\angle P + \angle \alpha = \angle C + \angle \beta \Rightarrow \angle \alpha - \angle \beta = \angle C - \angle P$
兩式相等：$\angle P - \angle B = \angle C - \angle P$
$2\angle P = \angle B + \angle C \Rightarrow \angle P = \frac{\angle B + \angle C}{2}$

二 飛鏢(凹四邊形)系列

模型 3：基本飛鏢
$\angle BDC = \angle A + \angle B + \angle C$
📝 展開證明
在 $\triangle ABD$ 中，外角 $\angle 5 = \angle 1 + \angle 3$
在 $\triangle ACD$ 中，外角 $\angle 6 = \angle 2 + \angle 4$
$\therefore \angle BDC = \angle 5 + \angle 6$
$= \angle 1 + \angle 2 + \angle 3 + \angle 4$
$= \angle B + \angle C + \angle A$
模型 4：內平分線
$\angle P = \frac{\angle A + \angle C}{2}$
📝 展開證明
由飛鏢模型可得：
小飛鏢：$\angle P = \angle \alpha + \angle A + \angle \beta$
$\Rightarrow \angle \alpha + \angle \beta = \angle P - \angle A$
大飛鏢：$\angle C = 2\angle \alpha + \angle A + 2\angle \beta$
$\Rightarrow 2(\angle \alpha + \angle \beta) = \angle C - \angle A$
兩式合併：$2(\angle P - \angle A) = \angle C - \angle A$
證得 $\angle P = \frac{\angle A + \angle C}{2}$
模型 5：上下平分線
$\angle P = \frac{\angle D - \angle B}{2}$
📝 展開證明
大飛鏢 $ABCD$：$\angle BCD = \angle B + \angle BAD + \angle D$
即 $2\angle \beta = \angle B + 2\angle \alpha + \angle D$
$\Rightarrow \angle \beta - \angle \alpha = \frac{\angle B + \angle D}{2}$
飛鏢 $ABCP$：$\angle BCP = \angle B + \angle P + \angle BAP$
即 $\angle \beta = \angle B + \angle P + \angle \alpha$
$\Rightarrow \angle \beta - \angle \alpha = \angle B + \angle P$
兩式相等：$\frac{\angle B + \angle D}{2} = \angle B + \angle P$
證得 $\angle P = \frac{\angle D - \angle B}{2}$

三 角平分線模型系列

模型 6：內角平分線
$\angle P = 90^\circ + \frac{1}{2}\angle A$
📝 展開證明
在 $\triangle ABC$ 中：$\angle A + 2\angle \alpha + 2\angle \beta = 180^\circ$
$\Rightarrow \angle \alpha + \angle \beta = 90^\circ - \frac{1}{2}\angle A$
在 $\triangle PBC$ 中：$\angle P + \angle \alpha + \angle \beta = 180^\circ$
$\Rightarrow \angle P = 180^\circ - (\angle \alpha + \angle \beta)$
代入得 $\angle P = 180^\circ - (90^\circ - \frac{1}{2}\angle A) = 90^\circ + \frac{1}{2}\angle A$
模型 7：外角平分線
$\angle P = 90^\circ - \frac{1}{2}\angle A$
📝 展開證明
※利用 A 字模型秒證：
$\angle MBC + \angle NCB = 180^\circ + \angle A$
$2\angle \alpha + 2\angle \beta = 180^\circ + \angle A$
$\Rightarrow \angle \alpha + \angle \beta = 90^\circ + \frac{1}{2}\angle A$
在 $\triangle BCP$ 中：$\angle P + \angle \alpha + \angle \beta = 180^\circ$
$\angle P = 180^\circ - (90^\circ + \frac{1}{2}\angle A) = 90^\circ - \frac{1}{2}\angle A$
模型 8：內外分模型
$\angle P = \frac{1}{2}\angle A$
📝 展開證明
$\angle ACM$ 是 $\triangle ABC$ 的外角：
$\angle ACM = \angle A + \angle ABC$
$\Rightarrow 2\angle \beta = \angle A + 2\angle \alpha$
$\Rightarrow \angle \beta - \angle \alpha = \frac{1}{2}\angle A$

$\angle PCM$ 是 $\triangle PBC$ 的外角：
$\angle PCM = \angle P + \angle PBC$
$\Rightarrow \angle \beta = \angle P + \angle \alpha \Rightarrow \angle P = \angle \beta - \angle \alpha$
證得 $\angle P = \frac{1}{2}\angle A$

四 A字模型與折角模型

模型 9：A 字模型
$\angle 2 + \angle 3 = \angle A + 180^\circ$
📝 展開證明
在 $\triangle ABC$ 中：
外角 $\angle 2 = \angle A + \angle ACB$
外角 $\angle 3 = \angle A + \angle ABC$

將兩式相加：
$\angle 2 + \angle 3 = \angle A + \angle ACB + \angle A + \angle ABC$
$= \angle A + (\angle A + \angle ABC + \angle ACB)$
$\because \triangle$ 內角和 $= 180^\circ$
$\therefore \angle 2 + \angle 3 = \angle A + 180^\circ$

模型 10：折角模型
折內：$\angle 1 + \angle 2 = 2\angle A$
折外：$\angle 1 - \angle 2 = 2\angle A$
📝 展開證明

【神級輔助線證明法】

連接 $\mathbf{AA'}$ 作輔助線。
以「折在角內」為例：
左邊 $\triangle$ 外角：$\angle 1 = \angle (\text{左}A) + \angle (\text{左}A')$
右邊 $\triangle$ 外角：$\angle 2 = \angle (\text{右}A) + \angle (\text{右}A')$

兩式相加：$\angle 1 + \angle 2 = \text{整顆}\angle A + \text{整顆}\angle A'$
因為摺疊，$\angle A = \angle A'$。
$\therefore \angle 1 + \angle 2 = 2\angle A$ (折在外同理，只是方向相反相減)。

五 垂直倒角系列

模型 11：高線與角平分線
$\angle DAE = \frac{1}{2}(\angle B - \angle C)$
📝 展開證明
$\because AD \perp BC \Rightarrow \angle DAB = 90^\circ - \angle B$
$\because AE$ 平分 $\angle BAC \Rightarrow \angle BAE = \frac{1}{2}(180^\circ - \angle B - \angle C) = 90^\circ - \frac{1}{2}\angle B - \frac{1}{2}\angle C$
夾角 $\angle DAE = \angle BAE - \angle DAB$
$= \frac{1}{2}\angle B - \frac{1}{2}\angle C = \frac{1}{2}(\angle B - \angle C)$
模型 12：雙垂直倒角
$\angle 1 = \angle 3, \quad \angle 2 = \angle 4$
📝 展開證明
條件：$AB \perp AD, AC \perp BD$
在 $\triangle ABC$ 中，$\angle 1 + \angle 2 = 90^\circ$
在 $\triangle ACD$ 中，$\angle 3 + \angle 4 = 90^\circ$
又 $\because \angle BAC + \angle CAD = 90^\circ$ 即 $\angle 2 + \angle 3 = 90^\circ$
$\therefore \angle 1 = 90^\circ - \angle 2 = \angle 3$
$\angle 4 = 90^\circ - \angle 3 = \angle 2$
模型 13：三垂直倒角 (K字)
$\angle 1 = \angle 3, \quad \angle 2 = \angle 4$
📝 展開證明
條件：$AB\perp BD, ED\perp BD, AC\perp CE$
在 $\triangle ABC$ 中，$\angle 1 + \angle 2 = 90^\circ$
在 $\triangle CDE$ 中，$\angle 3 + \angle 4 = 90^\circ$
直線 $BCD$ 上的平角：$\angle 2 + \angle 3 + 90^\circ = 180^\circ$
即 $\angle 2 + \angle 3 = 90^\circ$
$\therefore \angle 1 = 90^\circ - \angle 2 = \angle 3$
$\angle 4 = 90^\circ - \angle 3 = \angle 2$

實戰演練：原筆記經典題型解析

例 11. 三角板重疊問題

已知：直角頂點重合。$\angle B=90^\circ, \angle A=45^\circ, \angle D=30^\circ$。

求：$\angle AEF$ 的度數。

【解題步驟】

1. 外角定理：$\angle BFE = \angle A + \angle AEF$

2. 在 $\triangle BDF$ 中，$\angle BFE = 90^\circ - \angle D = 60^\circ$

3. $60^\circ = 45^\circ + \angle AEF \Rightarrow \mathbf{\angle AEF = 15^\circ}$

例 12. 正五角星角度和

已知：$\angle B, \angle E=27^\circ, \angle C, \angle D=31^\circ$。

求：$\angle A$ 的度數。

【解題步驟】

1. 將五角用「外角」集中到 $\triangle AFG$ 內：$\angle AFG = \angle C + \angle E$

2. $\angle A + (\angle C+\angle E) + (\angle B+\angle D) = 180^\circ$ (五角星內角和)

3. $\angle A = 180^\circ - 54^\circ - 62^\circ = \mathbf{64^\circ}$
